Key Features

🔬 RTL Design Analysis

Full SystemVerilog/VHDL design elaboration via Verific. Extracts all module parameters with resolved values, types, and constraints from the complete design hierarchy. Replaces error-prone manual LEC log parsing.

🗺 Memory Map → Instance Map

Consumes a structured address map spreadsheet (XLSX) and auto-generates instance map (.imap) files with base addresses, slot sizes, and RTL parameter overrides for every IP instance in the SoC.

🗄 Non-RTL Parameter Database

JSON-based ditabase for documentation-specific parameters (revision strings, security classifications, publishing modes) with chip_prefix-keyed hierarchical lookup across IP instances, SubSystems, and SoCs.

🤖 AI-Assisted Chapter Generation

Claude API extracts SoC-specific content (clocking, reset, interrupts, power management) from architecture specification PDFs and structures it as DITA concept/reference/task topics for human review.

📋 Register Diagram Rendering

Transforms register database (.rdb) files into DITA reference topics with interactive SVG bit-field diagrams and structured tables showing offset, field name, access type, and reset values.

🧠 RAG-Powered AI Chatbot

Built a retrieval-augmented generation knowledge base from resolved DITA. Users query the documentation in natural language. The chatbot grounds responses in actual spec content with source citations.
