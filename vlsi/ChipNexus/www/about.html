Our Story

Born from a Documentation Crisis

Modern SoCs contain 50+ IP blocks, each with hundreds of registers and thousands of configurable parameters. The traditional manual documentation process took 6–8 weeks per chip — and had a 15% error rate in parameter transcription.

We built the ChipNexus platform to solve this at the root. By connecting RTL design elaboration directly to documentation output, we eliminated the manual steps entirely. Today, the same SoC Reference Manual that took 8 weeks is generated in under 4 hours — with zero parameter errors.

Values
What Drives Us
🎯
Single Source of Truth
Every piece of data originates from one authoritative source and flows automatically — zero manual transcription, zero copy-paste errors.
🤖
AI as Assistant, Not Author
LLMs handle extraction and structuring. Humans retain creative authorship and final review authority. The machine proposes, the human disposes.
🔧
Tool-Agnostic Design
Every subsystem has swappable implementations. RTL Design Elaboration or Open-Source Elaboration. DITA-OT or Sphinx. Claude or GPT-4. Your toolchain, your choice.
🛡️
Defense in Depth
Multiple validation layers — RTL-to-documentation checking, address collision detection, schema validation — catch errors before publication.
🌍
Cross-Geo Standardization
Teams across continents use identical toolchains, eliminating format inconsistencies that require manual cleanup during integration.
⚡
Relentless Speed
Documentation should be generated overnight after RTL freeze. Same-day availability enables early software development and faster tape-outs.
Our Journey
Key Milestones
JUL 2022
Platform Inception
Identified the SoC documentation bottleneck. Began designing the automated pipeline architecture with RTL Design Elaboration elaboration integration.
JAN 2023
First Pipeline Prototype
End-to-end pipeline demo: RTL→parameters→IMAP→DITA→SVG diagrams. Proved the single-source-of-truth concept.
JUN 2023
VS Code Extension Launch
Released the unified VS Code extension with parameter explorer, build dashboard, and DITA preview. Lowered the barrier for non-CLI users.
DEC 2023
AI Chapter Generation
Integrated Claude API for extracting SoC-specific chapters from architecture PDFs. AI extracts, humans verify, pipeline includes.
MAY 2024
RAG Chatbot Launch
Flask-based RM Explorer with RAG-powered AI assistant. Reduced internal support queries by 40%.
SEP 2024
Non-RTL Parameter DB v3.0
CI-integrated gap analysis, contribution YAML workflow, schema validation. JSON becomes the single source for documentation configuration.
JAN 2025
50th Reference Manual Generated
Crossed 50 SoC Reference Manuals generated through the platform. Zero rollbacks to legacy process.
JUN 2026
Open-Source Release
ChipNexus platform released as open-source. XSD schemas published at bridgon.com/platform/. Modular architecture ready for community contributions.
